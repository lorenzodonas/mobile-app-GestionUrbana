# Gestión Urbana - Aplicación Móvil

Aplicación móvil desarrollada como parte del proyecto **Gestión Urbana**, cuyo objetivo es facilitar la comunicación entre la ciudadanía y la administración mediante la gestión de incidencias urbanas desde dispositivos móviles.

## Características

* Inicio de sesión de usuarios.
* Consulta de incidencias registradas.
* Creación de nuevas incidencias.
* Adjuntar imágenes a las incidencias.
* Consulta del estado de cada incidencia.
* Interfaz intuitiva adaptada a dispositivos móviles.

## Tecnologías utilizadas

* Flutter
* Dart
* API REST
* Base de datos MySQL (a través del backend)



## Instalación

1. Clonar el repositorio.
2. Ejecutar:

```bash
flutter pub get
```

3. Configurar la URL de la API.
4. Ejecutar la aplicación con:

```bash
flutter run
```

## Objetivo

Este proyecto fue desarrollado para aplicar conocimientos de desarrollo de aplicaciones móviles, consumo de APIs REST, autenticación de usuarios y diseño de interfaces modernas con Flutter.
